**Модуль** – это файл, содержащий определения функций, классов и переменных, а также исполняемый код. Модули в Python обычно имеют расширение .py. Они позволяют структурировать код и делить его на мелкие, легко управляемые части.


**Библиотека** — это набор связанных модулей, объединенных вместе для решения определенной задачи или предоставления определенного функционала. Библиотеки можно представить как расширение базового функционала Python, которое помогает разработчикам решать различные задачи без необходимости писать все с нуля.


Импорт модуля или библиотеки можно осуществить несколькими способами (рассмотрим на примере модуля math, подробнее о нем будет рассказано далее):
- Импорт всей библиотеки: import math
- Обращение к методам библиотеки будет происходить через ее название, например: print(math.pi) # Выведет число Пи
- Импорт всей библиотеки с присвоением ей псевдонима: import numpy as np
- Импорт отдельных методов и объектов в библиотеке: from math import pi
- Импорт всех методов из модуля/библиотеки: from math import *


Модули:
- Модуль math содержит большое количество функций для работы с числами, а также популярные константы Pi и e. 
- Модуль random позволяет генерировать случайные значения в заданном диапазоне.
- Модуль os предназначен для взаимодействия с операционной системой.
- Модуль time позволяет работать со временем системы.
